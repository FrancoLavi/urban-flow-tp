Urban Flow - Análisis de Multas por Exceso de Velocidad
Sprint actual: Sprint 1 - Análisis y procesamiento de datos
---

Objetivo
El objetivo de este sprint es trabajar sobre un dataset de infracciones por exceso de velocidad, realizando su carga, limpieza, normalización y análisis.

Se busca obtener un dataset consistente y confiable que permita generar métricas relevantes, identificar patrones y visualizar la información mediante gráficos.

---

Introducción y contexto
En este trabajo práctico se aborda el análisis de un conjunto de datos que contiene registros de infracciones de tránsito por exceso de velocidad.

Durante este sprint se desarrollan tareas fundamentales dentro del flujo de trabajo de análisis de datos, incluyendo:

Inicialización del proyecto y control de versiones con GitHub.
Organización de la estructura de carpetas para el manejo de datos.
Limpieza y normalización del dataset (fechas, horas, ubicaciones y patentes).
Eliminación de datos inválidos y outliers.
Generación de nuevas variables de análisis.
Implementación de una clase para encapsular y analizar la información.
Creación de visualizaciones que permitan interpretar los datos.

El enfoque del sprint está centrado en preparar y analizar los datos de manera estructurada, aplicando buenas prácticas de programación y análisis para facilitar su posterior utilización.
